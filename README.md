# 🤖 Luna Call Bomb & Management Bot

Bu proje, Telz API altyapısını kullanarak belirlenen numaralara otomatik doğrulama araması (auth call) gönderen ve çift bot sistemiyle çalışan bir Telegram otomasyonudur.

## 🚀 Özellikler
- **Çift Bot Sistemi:** Bir bot kullanıcı arayüzü, diğeri admin onay paneli olarak çalışır.
- **FSM Yönetimi:** Kullanıcı durum takibi (numara girişi, menü geçişleri).
- **Onay Sistemi:** Admin onayı olmadan kimse botu kullanamaz.
- **Otomatik Arama:** Belirli aralıklarla Telz API üzerinden arama tetikler.
